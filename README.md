# Crud using React + Vite

A simple CRUD (Create, Read, Update, Delete) project built with React + Vite.
This project simulates a fake doll shop where users can manage product data such as adding, viewing, and deleting items.

Feature
- Add new doll to the catalog
- View list of available dolls
- Delete dolls from the list
- Display doll images using URL
- Real-time UI updates using React state
