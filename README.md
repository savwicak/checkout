# Run Locally

Clone the project

```bash
git clone https://github.com/savwicak/checkout
```

Go to the project directory

```bash
cd checkout
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser and visit :DDD

```
http://localhost:5173/checkout
```
